Docker Native Workflow

Imagine the entire HYPER_ cloud as a single remote host with unlimited capacity. You can use the same Docker native workflow to deploy containers, as on your laptop.

Secure Container

HYPER_ leverages HyperContainer to apply the hardware-enforced isolation, giving you the security of virtual machines to containers with the speed of containers.

Persistent Volume / Snapshot

Run your stateful workload with HYPER_'s persistent volume. Multiple replicas will be automatically created with each volume to protect your data from failure. And you can take point-in-time snapshots of your volumes, for data backup/restore.

Free Network Traffic

All in/out/inter-container network traffic is FREE!

Floating IP

A floating IP is a publicly-accessible IP address that can be assigned to one container. Floating IPs can be remapped to other containers in the same region, to mask the failure of container or application.
