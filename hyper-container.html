What is HyperContainer

Hyper_ uses a technology called HyperContainer to apply hardware-enforced isolation to containers, while keeping the sub-second startup performance. Instead of running Cgroups and Namespaces, HyperContainer launches the Docker image using a hypervisor. This combination provides the best of both worlds, giving you the security of virtual machines with the speed of containers.

  • HyperContainer = Hypervisor + Docker Image

By combing the best from both virtualization and container world, HyperContainer is able to bring the hardware-enforced Isolation as VM, and the speed of container.
